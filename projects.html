2023

Procedurally Generated Music

A music generation project that makes music randomly. The structure of the piece, the notes played, and the sounds used are all generated randomly given a set of hard-coded constraints. No machine learning is involved. Still a work in progress... Check out the GitHub repo here.

McGill's McGameJam 2023: Raccoon Raider and the Temple of Trash

Game made for the McGameJam (hackthon kind of event). It's about a raccoon breaking stuff in a temple to find thrash. I made the music and most of the audio for it (the awesome voice lines are by Jack Xie). Check out the game here.

2022

Scène de la mer (Building 21 VR Piece)

A VR piece made within McGill's Building 21 collective that I made the music for. The piece was presented at Building 21's open-house on December 2022.

2022 McGill Hackthon: Sharkartack

This is a numerical simulation of the flocking of feeding sharks that some friends and I made for the 2022 McGill Hackthon. Check out the GitHub repo here.

Ising Model In JavaScript

An Ising Model simulation I made after having taken the statistical mechanics course at McGill. I made this as a motivation project while learning some JavaScript. Check out the GitHub repo here.
